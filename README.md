## Featured Projects

### Krili - Peer-to-Peer Rental Marketplace
A modern, full-stack rental marketplace designed to facilitate local sharing economies.
- **Stack**: Next.js 14, TypeScript, Tailwind CSS, Radix UI, MySQL, NextAuth v5
- **Key Features**: Secure multi-language support (English, French, Arabic with native RTL rendering), real-time direct messaging between users, dynamic listing management, responsive mobile-first UI, and custom analytics.
- **Scale**: Over 38,000 lines of code across 60+ pages and 100+ reusable components.

### EGOR - Multi-Model AI Backend & Hub
A FastAPI backend hosted on Hugging Face Spaces that handles advanced utilities and serves as an AI playground.
- **Stack**: FastAPI, Uvicorn, SQLAlchemy, MySQL (Aiven Cloud), Upstash Redis, TensorFlow
- **Key Features**: Multi-model chat hub integrating Google Gemini, Groq, Cerebras, and OpenRouter; a custom character-level LSTM neural network (REMAKA) for generating secure passwords; Fernet symmetric encryption; and sliding-window rate limiting.

### Light Pyrefly - VS Code Extension
A lightweight Python semantic highlighting extension for Visual Studio Code.
- **Stack**: TypeScript, VS Code Extension API
- **Key Features**: Standalone, context-aware syntax coloring that runs fast and without the resource overhead of heavy Python language servers.

### Other Projects
- **NEXUS-CoreTracker**: Real-time task manager built for team productivity.
- **Galactic Defender**: Space shooter game built with React.
- **FitMind**: Mental fitness tracker application.
